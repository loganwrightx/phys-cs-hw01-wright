# phys-cs-hw01-wright

## Problem 1

### Setup

1. Create a python venv with `python -m venv env`
2. Activate the `venv` and install dependencies with `source env/bin/activate && pip install -r requirements.txt`
3. Run the benchmark program with `cd problem1/part_a && python bench_nbody.py` – see the generated log-log plot generated as a PNG image file. The numerical values of the fitted line are also printed to the console upon completion of the program in both the linear form and the asymptotic formula demonstrating that time complexity approaches $O(N^2)$ when $N$ is large.
